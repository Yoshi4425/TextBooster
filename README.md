## TextBooster version2.0 概要
従来の「定型文貼り付けツール」から、**「膨大なストックを効率的に管理・運用するナレッジベース・ランチャー」**へと進化しました。SQLiteデータベースの導入により、数千件のデータの中から必要なものだけを30個のスロットに呼び出して使用できます。
### 主なアップデート項目
データ永続化の強化: SQLiteによる高速な検索と安定した保存。
インテリジェントHUD: 入力した数字を美しく透過表示し、視認性を極限まで向上。
30番スロットの専用化: システム設定（管理画面）への専用アクセス口を設置。
ウィンドウ吸着システム: 管理画面がメインパネルの隣に自動整列するスマートなレイアウト。

## 操作方法マニュアル
###1. メインパネルの基本操作メインパネルには現在スロットに割り当てられている29種類の定型文と、設定ボタンが表示されます。
数字入力による貼り付け:キーボードで 「01」〜「29」 の数字を入力し、Enterキー を押すと、該当するスロットの本文がクリップボードにコピーされ、自動で貼り付け（Ctrl+V）が実行されます。
HUD表示:入力中の数字は画面中央に巨大なオーバーレイ（HUD）として表示されます。2秒間入力がない、またはEnterを押すと自動的に消去されます。
設定画面の起動:30番スロット（⚙ 設定画面） をクリックするか、キーボードで 「30」➔ Enter と入力すると、データベース管理画面が起動します。
### 2. データベース管理画面（ClipDatabaseWindow）設定ボタンを押すと、メインパネルの左隣にピタッと吸着して管理画面が開きます。
#### データの一覧と検索検索パネル: キーワード、大ラベル、小ラベルの組み合わせで、ストックされた全データを瞬時にフィルタリングできます。
テーブル表示: 「タイトル（40%）」「大ラベル（30%）」「小ラベル（30%）」の比率で見やすく表示されます。
#### スロットへの割り当て（重要）一覧からパネルに表示したいデータを選択します。右側の詳細編集パネルにある 「Slot(1-29)」 欄に、配置したい番号を入力します。「確定」 ボタン（またはEnter）を押すと、メインパネルの該当番号にそのデータが即座に配属されます。注意: Slot 30 はシステム専用のため、割り当てることはできません。
#### データの編集・新規登録新規: 入力フォームを埋めて「新規」を押すと、新しい定型文がデータベースに追加されます。更新: 既存のデータを選択して内容を書き換え、「更新」を押すと保存されます。削除: 不要になったデータをデータベースから完全に削除します。

## 仕様一覧項目内容データ保存SQLite 3 (DatabaseHelper.java)
最大スロット数29スロット（+1設定専用ボタン）
HUD仕様透過度180 / 等幅フォント / 自動リフレッシュ
ウィンドウ配置メインパネル左端への自動吸着 (Snapping)
検索比率タイトル:大ラベル:小ラベル = 4:3:3

開発者メモ:スロット割り当ての際、0 を入力して確定すると、そのデータのスロット配属を解除（スロットから外す）することができます。常に使うものは 1〜10、たまに使うものは管理画面からその都度アサインする、といった動的な使い方がオススメです。
